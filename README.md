# g-overdose
Overdose script

## items
- Joint
- Heroin
- Lsd
- Bongo
- Tramadol
- Heroin
- Morphine
- Adrenaline
- Meldonin
- Ketamine

## Installation
1. Clone or download this repo
2. Copy the `g-overdose` folder to your server resources folder
3.  Add `start g-overdose` to your server.cfg
4. Or add `ensure g-overdose` to your server.cfg
5. Restart the server
